Desafios da automação tradicional.

⏳

Setup Demorado

Tempo inicial de configuração do projeto pode levar dias, incluindo Frameworks, dependências, estrutura de pastas… antes de escrever um único teste.

8-16h por projeto

Manutenção Constante

XPaths quebram a cada deploy. Classes CSS mudam. O time gasta mais tempo consertando seletores do que testando funcionalidades.

60% do tempo gasto em manutenção

login_page.py 
class LoginLocators:
    USUARIO    = (By.CSS_SELECTOR, "#username")
    SENHA      = (By.CSS_SELECTOR, "#password")
    BTN_ENTRAR = (By.XPATH, "//button[@type='submit']")

class LoginPage(BasePage):
    def preencher_usuario(self, texto):
        """Preenche o campo usuario"""
        return self._fill(self.locators.USUARIO, texto)

    def clicar_botao_entrar(self):
        """Clica no botão entrar"""
        return self._click(self.locators.BTN_ENTRAR)
