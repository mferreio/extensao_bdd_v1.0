Política de Privacidade - BDD Generator

Última atualização: 1 de julho de 2025
Desenvolvedor: Matheus Ferreira de Oliveira

1. Introdução

Esta Política de Privacidade descreve como a extensão "BDD Generator" coleta, usa e protege suas informações quando você utiliza nossa extensão do Chrome para geração automatizada de testes BDD (Behavior-Driven Development).

2. Informações que Coletamos

Nossa extensão coleta apenas as seguintes informações, exclusivamente de forma local:

  • Atividade do usuário: Interações com páginas web (cliques, digitação, navegação) apenas enquanto o modo de gravação estiver ativo
  • Estrutura da página: Elementos HTML e seus seletores para geração de código de teste
  • URLs visitadas: Apenas para registrar passos de navegação nos testes
IMPORTANTE: Todos os dados são processados apenas localmente no seu navegador. Nenhuma informação é enviada para servidores externos ou terceiros.

3. Como Usamos suas Informações

As informações coletadas são usadas exclusivamente para:

  • Gerar código de teste automatizado em Python/Selenium
  • Criar arquivos .feature no formato Gherkin
  • Produzir page objects e steps para automação de testes
  • Melhorar a precisão dos seletores CSS/XPath gerados

4. Armazenamento de Dados

  • Local: Todos os dados são armazenados apenas no seu navegador
  • Temporário: Os dados são mantidos apenas durante a sessão de gravação
  • Controle total: Você pode limpar todos os dados a qualquer momento usando o botão "Limpar" na extensão
  • Sem backup: Não fazemos backup ou sincronização dos seus dados

5. Compartilhamento de Dados

NÃO compartilhamos, vendemos ou transferimos suas informações para terceiros.

A extensão:

  • Não envia dados para servidores externos
  • Não usa serviços de analytics
  • Não implementa tracking ou cookies
  • Não se conecta com APIs externas

6. Segurança

Implementamos as seguintes medidas de segurança:

  • Processamento 100% local dos dados
  • Não armazenamento de dados sensíveis
  • Código fonte auditável e transparente
  • Permissões mínimas necessárias

7. Seus Direitos

Você tem total controle sobre seus dados:

  • Acesso: Visualize todas as ações capturadas no painel da extensão
  • Exclusão: Use o botão "Limpar" para remover todos os dados
  • Controle: Pause ou pare a gravação a qualquer momento
  • Exportação: Exporte seus dados quando desejar

8. Dados Sensíveis

AVISO IMPORTANTE: Evite gravar testes em páginas com informações sensíveis como:

  • Senhas reais
  • Números de cartão de crédito
  • Documentos pessoais
  • Informações confidenciais

Use sempre dados de teste em ambientes controlados.

9. Menores de Idade

Nossa extensão não é direcionada para menores de 13 anos. Não coletamos intencionalmente informações de crianças menores de 13 anos.

10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade ocasionalmente. Alterações significativas serão comunicadas através de atualizações da extensão.

11. Conformidade

Esta política está em conformidade com:

  • Políticas do Chrome Web Store
  • LGPD (Lei Geral de Proteção de Dados)
  • GDPR (General Data Protection Regulation)
