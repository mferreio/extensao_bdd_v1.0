Manual do Usuário

Bem-vindo ao Assistente Automatizador! Esta ferramenta ajuda você a criar testes automatizados em Python de forma rápida e intuitiva, gravando suas ações no navegador.

🚀 Como Usar a Ferramenta

1. Criando uma Feature

1
Ao abrir a extensão, digite o nome da sua Feature (funcionalidade macro que será testada e.g., "Login", "Carrinho").
2
Clique em "Criar Feature".

2. Gravando um Cenário

3
Dê um nome para o seu Cenário (caso de teste específico e.g., "Login com sucesso", "Senha incorreta").
4
Clique em "Iniciar Gravação". A partir de agora, suas ações na página serão capturadas.
5
Interaja com a página:
  • Clique em botões e links.
  • Preencha campos de formulário.
  • Use a tecla TAB ou clique fora para confirmar o preenchimento de campos.
Dica: Para validar se um elemento, texto ou imagem está na tela, ative o modo "Inspecionar Elemento" (ícone de olho), clique no elemento desejado e depois escolha a validação na lista.

3. Finalizando e Exportando

6
Quando terminar o fluxo, abra a extensão e clique em "Parar Gravação".
7
Você pode adicionar mais cenários à mesma feature ou finalizar a feature atual.
8
Quando tudo estiver pronto, clique em "Finalizar e Exportar". Um arquivo projeto_bdd.zip será baixado.

💻 Como Rodar os Testes (Python)

O arquivo ZIP exportado contém uma estrutura completa de projeto. Siga os passos abaixo para executá-lo:

Pré-requisitos

  • Python instalado.
  • Navegador Google Chrome instalado.

Passo a Passo

  1. Descompacte o arquivo projeto_bdd.zip em uma pasta.
  2. Abra o terminal (CMD ou PowerShell) e navegue até essa pasta: 
    cd caminho/para/sua/pasta
  3. Instale as dependências (necessário apenas na primeira vez): 
    pip install -r requirements.txt
  4. Execute os testes: 
    behave
O navegador será aberto automaticamente e executará os passos que você gravou. Ao final, um relatório será exibido no terminal.
